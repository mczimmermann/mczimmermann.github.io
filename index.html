I'm a master's student at SecLab at UCSB, graduating in December 2025. I'm interested in the intersection of security and machine learning. Feel free to check out my projects, research, or get in touch! 🚀

Projects

DARPA AI Cyber Challenge
Working as part of a semi-finalist team to develop AI-driven tools for identifying and autonomously patching software vulnerabilities. Currently, designing and implementing a system of multiple LLM agents that create a feedback loop to find the crashing inputs for vulnerable functions.

Bedrock: Security for Smart Contracts
Wrote an algorithm to learn features of smart contracts at the Ethereum Virtual Machine (EVM) three-address code (TAC) level, and return vectorized forms of the contracts. The vectors were then used to train a machine learning classifier to detect Ponzi schemes.
We presented the project and poster at the Early Research Scholars Program National Conference during September 2023, which features works from universities around the U.S.
